# 简单-白红模板 - 设计规范

## I. 模板概述

**模板名称**: 简单-白红 (Simple White-Red)
**适用场景**: 企业内部汇报、工作总结、业务介绍、项目汇报
**设计风格**: 简洁、专业、现代，以白色为主背景，红色为强调色
**设计基调**: 清晰、直接、商务专业

## II. 画布规格

- **画布格式**: PPT 16:9
- **尺寸**: 1280 x 720 像素
- **viewBox**: `0 0 1280 720`
- **出血区**: 无

## III. 配色方案

| 颜色角色 | 色值 | 用途 |
|----------|------|------|
| **主色（红色）** | `#C8102E` | 标题、强调元素、装饰线条、重点标记 |
| **辅助色（深红）** | `#A00C24` | 悬停状态、次要强调 |
| **背景色** | `#FFFFFF` | 主背景 |
| **浅灰背景** | `#F5F5F5` | 内容区域背景、卡片背景 |
| **主文字** | `#1A1A1A` | 正文、标题 |
| **次要文字** | `#666666` | 辅助文字、说明文字 |
| **浅色文字** | `#999999` | 页码、日期、装饰性文字 |
| **边框/分割线** | `#E0E0E0` | 分隔线、边框 |

## IV. 字体系统

- **标题字体**: Microsoft YaHei, PingFang SC, sans-serif
- **正文字体**: Microsoft YaHei, PingFang SC, sans-serif
- **英文/数字**: Arial, Helvetica, sans-serif

### 字号层级

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| **主标题** | 48-56px | Bold | 封面标题 |
| **副标题** | 24-32px | Regular | 封面副标题 |
| **章节号** | 72-96px | Bold | 章节页数字 |
| **章节标题** | 36-42px | Bold | 章节页标题 |
| **页面标题** | 28-32px | Bold | 内容页标题 |
| **正文大** | 18-20px | Regular | 重点内容 |
| **正文常规** | 14-16px | Regular | 普通正文 |
| **注释** | 12px | Regular | 页码、日期、来源 |

## V. 页面结构

### 通用布局原则

- **页边距**: 左右 60px，上下 50px
- **内容区**: 宽度 1160px，高度 620px
- **网格系统**: 12列网格，列间距 20px
- **对齐**: 左对齐为主，标题可居中

### 装饰元素

- **红色装饰条**: 宽度 4-6px，用于左侧或顶部强调
- **红色底线**: 标题下方 2-3px 红色底线
- **几何图形**: 简洁的矩形、线条装饰

## VI. 页面类型

### 1. 封面页 (01_cover.svg)

- 白色背景
- 左侧红色装饰条（宽 6px，高 200px）
- 主标题居中偏上
- 副标题在主标题下方
- 日期和作者信息在底部

### 2. 章节页 (02_chapter.svg)

- 白色背景
- 大号章节数字（红色，透明度 0.15）作为背景装饰
- 章节标题（黑色）
- 红色底线装饰

### 3. 目录页 (02_toc.svg) [可选]

- 白色背景
- 页面标题
- 目录项列表，带红色序号标记

### 4. 内容页 (03_content.svg)

- 白色或浅灰背景
- 顶部标题栏（带红色底线）
- 主内容区域（灵活布局）
- 底部页码

### 5. 结束页 (04_ending.svg)

- 白色背景
- 感谢文字居中
- 联系信息在底部
- 红色装饰元素

## VII. 布局模式推荐

- **左右分栏**: 适用于图文混排
- **上下分区**: 适用于标题+内容
- **卡片式**: 适用于多要点展示
- **全宽布局**: 适用于大段文字或图表

## VIII. 间距规范

- **大间距**: 40-60px（区块之间）
- **中间距**: 20-30px（元素组之间）
- **小间距**: 8-12px（相关元素之间）
- **行高**: 1.5-1.8（正文）

## IX. SVG 技术约束

- **viewBox**: 必须严格使用 `0 0 1280 720`
- **禁用特性**: clipPath, mask, `<style>`, class, external CSS, foreignObject, textPath, @font-face, animate, script, marker-end, iframe
- **透明度**: 使用 `fill-opacity` / `stroke-opacity` 替代 rgba()
- **字体**: 使用系统字体，不嵌入自定义字体
- **图片**: 如需图片，使用 `<image>` 标签引用外部资源

## X. 占位符规范

| 占位符 | 用途 | 适用页面 |
|--------|------|----------|
| `{{TITLE}}` | 主标题 | 封面 |
| `{{SUBTITLE}}` | 副标题 | 封面 |
| `{{DATE}}` | 日期 | 封面 |
| `{{AUTHOR}}` | 作者/组织 | 封面 |
| `{{CHAPTER_NUM}}` | 章节数字 | 章节页 |
| `{{CHAPTER_TITLE}}` | 章节标题 | 章节页 |
| `{{PAGE_TITLE}}` | 页面标题 | 内容页 |
| `{{CONTENT_AREA}}` | 内容区域 | 内容页 |
| `{{PAGE_NUM}}` | 页码 | 内容页、结束页 |
| `{{THANK_YOU}}` | 感谢文字 | 结束页 |
| `{{CONTACT_INFO}}` | 联系信息 | 结束页 |
| `{{TOC_ITEM_1_TITLE}}` | 目录项1标题 | 目录页 |
| `{{TOC_ITEM_1_DESC}}` | 目录项1描述 | 目录页 |

## XI. 设计原则

1. **简洁至上**: 避免过多装饰，保持页面清爽
2. **红色点睛**: 红色仅用于强调，不宜大面积使用
3. **层次分明**: 通过字号、颜色、间距建立清晰的视觉层级
4. **留白充足**: 保持足够的呼吸空间，避免拥挤
5. **专业稳重**: 适合商务场景，传达专业可靠的形象
